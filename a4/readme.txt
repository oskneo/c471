as4.sh is the bash script to run on any host at netlab that can create mgn scripts and other needed bash scripts to run the experiment.

Example:
    as4.sh
    This command will use our default address and port 224.5.5.5 5678.
    as4.sh 224.5.5.5 5678
    This command can accept two auguments that the first one is group address.
    And the second one is the port.
